# **App Name**: LearnScape

## Core Features:

- Student Class Selection & Quiz Flow: A dynamic landing page for students to select a class via a scroll-down menu, input their name, take interactive math quizzes, and review results including submitted answers, correct solutions, detailed problem-solving steps, a final score, a 'Selesai' (Done) button, and a 'Return to Home' option.
- Teacher Secure Access: A discreet login for teachers via the application's logo/text, secured with the password 'admin123', providing access to the administrative dashboard, along with an option to return to the student view.
- Teacher Content Management: Comprehensive CRUD capabilities for teachers to manage classes (create, read, update, delete) and associated math questions, including problem statements, answer choices, and solution methods.
- Student Performance Tracking: A dashboard view for teachers to monitor student quiz results, displaying overall statistics like total participants, average scores, recent activity, and score distribution per class.
- Adaptive Theming: Provides users with options to switch between light, dark, and system-preferred display themes for an optimized viewing experience across different environments.
- Engaging User Animations: Incorporates modern and attractive animations for smooth transitions between pages, interactive elements, and dynamic data presentations to enhance overall user engagement and visual appeal.

## Style Guidelines:

- Primary color: A vibrant yet professional blue (#457CC4) for main interactive elements and branding, conveying trust and focus.
- Background color: A subtle light blue-gray (#F0F2F5) providing a clean, spacious backdrop for content in the light theme.
- Accent color: An energetic cyan-blue (#26B3D6) used for highlighting key information, call-to-action buttons, and to create visual contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly techy, and engaging feel. Body font: 'Inter' (sans-serif) for high readability and a clean, neutral presentation of text content.
- Utilize a consistent set of modern, minimalist icons that clearly represent their functions, enhancing usability and matching the clean aesthetic.
- Implement a responsive design with a clean, modular layout. The student landing page will feature a clear scroll-down class selection, while the teacher dashboard will use a sidebar navigation and card-based data presentation for clarity.
- Subtle yet modern animations for state changes, form submissions, navigation between pages, and interactive components like button hovers and class selections, to provide a polished and engaging user experience.
